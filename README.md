# ADAPT Framework Template
**Densight Labs — Applied AI. Not just talked about.**

A structured template for planning and executing AI projects using Densight Labs' ADAPT Framework. Built for practitioners who need clarity on scope, stakeholders, success metrics, and deployment paths before writing a single line of code.

## What Is Inside
- Phase-by-phase project breakdown (Assess, Design, Architect, Prototype, Train)
- Stakeholder mapping and decision matrix
- Success metrics definition worksheet
- Risk assessment checklist

## ADAPT Phase: Assess
This template lives in the Assess phase—where you define the problem, map stakeholders, and set success criteria before solution design begins.

## Template
**Use this when:** Starting any AI project that needs structured planning and clear deliverables.

**Key Fields:**
- Problem Statement
- Stakeholder Map (decision makers, influencers, end users)
- Success Metrics (business KPIs, technical benchmarks)
- Constraints (budget, timeline, data availability)
- Risk Register
- Initial Scope Boundary

**Example:**
*Problem:* Customer support response time exceeds 24 hours.
*Stakeholder:* CS Director (decision), Support Team (end user).
*Success Metric:* Reduce response time to <4 hours within 90 days.
